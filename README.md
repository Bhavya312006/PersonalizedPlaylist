# PersonalizedPlaylist

A personally crafted, aesthetic music player web app with a magical petal cursor, playlist folders, and real YouTube playback. Built for personal use - no logins, no ads, just vibes.

Features

  - Magical cursor — petals and flowers trail your mouse across the screen
  - Real YouTube playback — paste any YouTube link to play songs (any language — Hindi, Tamil, Telugu, K-pop, English, and more)
  - Create playlists — make custom folders and add songs into them
  - Like songs — save your favourites
  - Full player controls — play, pause, skip, shuffle, repeat, volume
  - Song poster — YouTube thumbnail shows as the album art, spins like vinyl
  - Auto-saves — your songs and playlists stay saved even after closing the browser
  - I have creted it using python !!
